## Development Setup

**Clone this project**

```bash
# Move to your workspace
cd your-workspace

# Clone this repository:
$ git clone https://github.com/whl5105/flutter_lotto.git

# Enter in directory:
$ cd flutter_lotto

# For install dependencies:
$ flutter pub get

# Run the app:
$ flutter run

```
